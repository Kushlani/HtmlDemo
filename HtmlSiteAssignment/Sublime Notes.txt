4. Install tidy

1. Download Zip (http://binaries.html-tidy.org/) x64
https://github.com/htacg/tidy-html5/releases/download/5.6.0/tidy-5.6.0-vc14-64b.zip

2. Extract into a location (C:\Development\Software)
3. Go inside the unzipped folder bin file and copy the path 
	(C:\Developement\Software\tidy-5.6.0-vc14-64b\bin)
4. Then add the path to the PATH varibale.
This Pc > Properties > Advanced System Settings > Envrironment Variables
System Variables select Path variable, edit add the copied path followed by  ;
